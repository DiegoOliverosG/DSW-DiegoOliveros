Primero, lo primero: Entendiendo qué es HTML


Para abordar este tema es preciso primero apropiarse del concepto de HTML. HTML (Lenguaje de Marcas de Hipertexto, del inglés HyperText Markup Language) Es un código que se usa para estructurar una página web y lo que esta contiene, ya sean párrafos, viñetas, imágenes, tablas, etc. Y de ese modo poder hacer que se visualicen o se comporten de determinada manera. El autor Castillo (2007) lo define de la siguiente manera:

El HTML es un lenguaje de marcas diseñado para estructurar textos y presentarlos en forma de hipertexto, que es el formato estándar de las páginas Web. Gracias a Internet y a los navegadores del tipo Explorer o Netscape, el HTML se ha convertido en uno de los formatos más populares que existen para la construcción de documentos. (p. 14)
