Etiquetas básicas de HTML

Continuando con esta temática, Las etiquetas de HTML no son más que pequeños bloques de código que tienen unas funciones y unos usos específicos empleados para escribir código en HTML, señalando al navegador cómo debe interpretarlas en cuanto a su contenido; es decir si se quiere insertar un texto en forma de párrafo, hay una etiqueta específica para que el navegador pueda interpretar ese texto como un párrafo. La estructura de las etiquetas HTML comienza con el símbolo «<» y termina con el símbolo «>». En la mitad de estos dos símbolos va el nombre de la etiqueta que se quiere que interprete el navegador. Ahora se van a conocer ejemplos de estas etiquetas, así que permaneciendo con los conceptos que brinda Castillo (2007) Las etiquetas básicas de HTML, de obligada presencia en todo documento son:

HTML:

Es la etiqueta que define el inicio del documento HTML, le indica al navegador que todo lo que viene a continuación debe tratarlo como una serie de códigos HTML.

HEAD:

Define la cabecera del documento HTML, esta cabecera suele contener información sobre el documento que no se muestra directamente en el navegador. Como por ejemplo el título de la ventana de su navegador. Dentro de esta etiqueta podemos encontrar:

TITLE:

Define el título de la página. Por lo general, el título aparece en la barra de título encima de la ventana.

LINK:

Para definir algunas características avanzadas, como por ejemplo los estilos usados para la visualización de la página

BODY

Define el contenido principal o cuerpo del documento, ésta es la parte del documento HTML que se muestra en el navegador, dentro de esta etiqueta pueden definirse propiedades comunes a toda la página, como color de fondo y márgenes. Dentro del cuerpo BODY podemos encontrar numerosísimas etiquetas. A continuación, se indican algunas a modo de ejemplo:

H1, H2..., H6:

Encabezados o títulos del documento en diferentes tamaños de fuente.

P:

Párrafo nuevo.

BR:

Salto de línea forzado.

TABLE:

Comienzo de una tabla (las filas se identifican con y las celdas dentro de las filas con ).

a:

Indica la existencia de un hipervínculo o enlace, dentro o fuera la página Web. Debe definirse el parámetro href que indica a donde apunta en enlace.

DIV:

Comienzo de un área especial en la página. (p. 14-15)

Atributos

Entender el concepto de atributo en HTML es tan fácil como comprender que es la propiedad que se le agrega a una etiqueta y así proporciona más información sobre ese elemento. Según Sanchez (2017) “Este otorgará una propiedad o comportamiento específico a la propia etiqueta.” (párr. 2) Por tal motivo es importante tener en cuenta los diferentes atributos que existen, a continuación, un pequeño listado los atributos más comunes:

id:

Establece un identificador a la etiqueta HTML (sólo una por página con ese nombre).

Class:

Establece una clase (género) a una etiqueta HTML. Puede repetirse por página.

Lang:

Indica el idioma del contenido de la etiqueta HTML.

Translate:

Indica si el contenido de la etiqueta es traducible o no.

title:

Mensaje mostrado en un tooltip (aviso amarillo emergente) al mover el ratón encima.

data-*:

Metadatos en la propia etiqueta. Se puede usar cualquier nombre con prefijo data-.

Accesskey:

Combinación de teclas que puede pulsar el usuario para activar el elemento.

dir:

Establece la direccionalidad del texto (left to right o right to left).

Style:

Aplica propiedades CSS directamente al elemento en cuestión.

Valor

Este término hace referencia a como su nombre lo indica, al valor primitivo que contiene el contenedor de objetos; entendiendo valor primitivo como los datos que no son un objeto y no tienen métodos.
